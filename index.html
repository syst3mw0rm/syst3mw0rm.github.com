ownCloud (May 2011 - August 2011)
Season of KDE 2011

ownCloud is a flexible, open source file sync and share solution. I worked on synchronization of two OwnCloud server nodes. I essentially attacked the problem in three ways:

  • Trivial method : Copying the complete file, not so optimal in case small changes are made to some big files.
  • Using rsync to take care of synchronization.
  • Using glip (https://github.com/patrikf/glip) to store files in form of git commits and transfer only the commit data when files are modified.
